Projeto desenvolvido para a disciplina de Programação Concorrente e Paralela.  

Aplicação de Sockets em um sistema cliente-servidor de renderização de imagem com suavização de stencil de 5 pixels.  

O projeto envolve a suavização de imagens fictícias, representadas por uma matriz de inteiros. 

# 

Consiste em um sistema cliente-servidor, onde o servidor possui uma imagem (imagem falsa, somente uma matriz de inteiros representando os pixels) que será dividida entre os N clientes.  

Os clientes retornam seus pedaços ao servidor, que escreve a imagem renderizada em um arquivo.  

# 

Para desenvolver foi utilizado a linguagem C e bibliotecas de sockets.
